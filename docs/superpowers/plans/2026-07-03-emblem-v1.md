# Emblem v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Emblem — a macOS app that gives Finder sidebar Favorites custom SF Symbol icons — and install it working on this machine.

**Architecture:** Hybrid port of ivg-design/SidebarFavorites: the icon-app generation engine (template copy → plist mutation → codesign extension-first → lsregister) ports nearly verbatim into an `EmblemCore` framework; the SwiftUI shell (list + status badges, symbol browser, 4-step setup checklist, menu bar) is written fresh with `@Observable`.

**Tech Stack:** Swift 5.9+/SwiftUI, macOS 14+ deployment, Xcode 26.6, XcodeGen, XCTest, GitHub Actions.

## Global Constraints (from spec, verbatim)

- Minimum macOS: **14** (Sonoma). Build with Xcode 26.6 SDK; gate any macOS-26-only API with `#available`.
- Bundle IDs: app `page.klee.emblem`; generated icon apps `page.klee.emblem.icon.<uuid>`, extension `page.klee.emblem.icon.<uuid>.sync`.
- Storage: `~/Library/Application Support/Emblem/{config.json, Apps/, Icons/}`.
- Engine sequence is a contract (spec §2): copy template → (custom SVG only) actool compile → mutate plists + FolderPath.txt → sign **extension first** with sandbox entitlements then app → `lsregister -f -R -trusted`. Removal: `pluginkit -e ignore -i` → `lsregister -u` → delete.
- Signing resolution: Automatic → prefers "Apple Development", then "Developer ID Application", else ad-hoc `-`.
- All subprocess calls async off the main actor; UI never blocks.
- Deep link: `x-apple.systempreferences:com.apple.ExtensionsPreferences?extension-points` (fallback: same without query).
- MIT LICENSE must retain the ivg-design copyright line.
- Commit style (user rule): one line, < 50 chars, no Co-Authored-By.

Rules: tilde-expand; resolve symlinks for the *cloud* check only (`URL.resolvingSymlinksInPath()`); `cloudStorage` if resolved path has `<home>/Library/CloudStorage` prefix; `tccProtected` if the path *itself* (unresolved) is `<home>/Desktop`, `<home>/Documents`, or `<home>/Downloads` or a descendant; else `normal`.
